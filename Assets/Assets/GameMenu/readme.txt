EN
The asset contains two scenes: MainMenu, where the game menu is directly presented, and Scene01 to demonstrate the ability to load the game.
To add your own sounds to the game and adjust their volume, you need to place the sound file in the Sound directory and drag it to the stage. don't forget to specify the group for it in the Output section of the Audio Source component: EffectGroup for sound effects and Misispoir for music.
To change the appearance of buttons, upload your button image file to the Sprites - Interface folder


РУ
Ассет содержит две сцены: MainMenu, где непосредственно представлено меню игры и Scene01 для демонстрации возможности загружать игру.
Для добавления своих звуков в игру и возможности регулирования их громкости, нужно расположить звуковой файл в каталог Sound и перетащить на сцену не забыв указать для него в разделе Output компонента Audio Source группу: EffectGroup для звуковых эффектов и MusicПroup - для музыки.
Чтобы изменить внешний вид кнопок, необходимо загрузить свой файл изображения кнопки в папку Sprites - Interface
